## 2. 업무 요청 — 실사 상품 이미지

### 문제
10차까지 `public/products/*.svg` 벡터 아이콘을 사용해, 쇼핑앱 느낌의 **실제 상품 사진**이 아니었음.

### 조치

| ID | 품목 | 이미지 |
|----|------|--------|
| 1 | 3겹 화장지 30롤 | `/products/1.webp` — 화장지 롤 묶음 실사 |
| 2 | 액체 세탁세제 3L | `/products/2.webp` — 세탁세제 병 실사 |
| 3 | 주방세제 1L 2개 | `/products/3.webp` — 주방세제 병 실사 |
| 4 | 샴푸+린스 세트 | `/products/4.webp` — 샴푸·린스 세트 실사 |
| 5 | 물티슈 80매 10팩 | `/products/5.webp` — 물티슈 팩 실사 |

- **로컬 저장:** 외부 CDN 없이 `public/products/`에 호스팅 → 배포 환경에서도 안정적 로드
- **WebP 압축:** 800×600, quality 82 → 파일당 약 8~22KB (기존 SVG 대비 실사 품질 + 빠른 로딩)
- **SVG 제거:** 벡터 상징 파일 전부 삭제

### 적용 화면
- 홈·탐색 카드 (`GroupBuyCard` / `ProductImage`)
- 공구 상세 `/detail/:id` 히어로
- 결제 `/payment/:id` 썸네일
- 공구 열기 `/open` 카탈로그 선택·step2·step3
