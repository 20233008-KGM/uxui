## 3. 이번 작업: `/screens` 데스크탑 최적화

### 변경 전 문제
- 가로 한 줄 endless scroll (모바일 와이어프레임 관점)
- `#root` 중앙 정렬로 넓은 화면 활용 불가
- 데스크탑에서 섹션 탐색·줌 불편

### 변경 후 (`ScreenCanvasPage.tsx`)

| 기능 | 설명 |
|------|------|
| **전체 화면** | `fixed inset-0` — 데스크탑 뷰포트 100% 사용 |
| **반응형 그리드** | 1~5열 (768 / 1280 / 1536 / 1920px 기준) |
| **좌측 사이드바** | lg+ 섹션 점프 네비 + 화면 수 표시 |
| **화면 맞춤** | 창 너비·열 수에 맞춰 자동 스케일 |
| **키보드** | `+`/`−` 줌, `0` 맞춤, Ctrl+휠 줌 |
| **sticky 헤더** | 섹션 제목 스크롤 시 고정 |
| **최대 너비** | 2400px 중앙 정렬 (울트라와이드) |
