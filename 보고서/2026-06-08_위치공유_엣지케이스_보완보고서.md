## 2. 신규 구현: 팀장 위치 공유 · 거리 기반 탐색

### 2.1 공구 열기 (팀장)
| 기능 | 위치 |
|------|------|
| 수령 위치 선택 UI | `/open/step2` |
| 지도형 미리보기 + 프리셋 (신림동, 신림역, 봉천동, 남현동) | `LocationPicker` 컴포넌트 |

### 2.2 참여자 탐색 · 상세
| 기능 | 설명 |
|------|------|
| 거리 배지 | 카드·상세에 `350m`, `1.2km` 표시 |
| 동네 범위 필터 | 탐색/슬라이드 — 설정 범위(500m~동) 밖 공구 제외 |
| 가까운 순 정렬 | 거리 오름차순 |
| 거리 경고 | 상세 — 설정 범위 초과 시 `DistanceWarning` |
| 공구 정보 | 수령 위치(팀장 공유), 나와의 거리 표시 |

## 6. 확인 방법

```bash
npm run dev
```

1. **위치 공유:** `/open` → 2단계에서 수령 위치 선택 → 3단계에서 확인
2. **거리 탐색:** `/explore` 또는 `/explore/slide` — 거리 배지 확인
3. **멀리 있는 공구:** `/detail/5` (남현동) — 거리 경고 확인
4. **범위 필터:** 마이 → 동네 범위 500m로 줄이면 목록 감소
5. **엣지케이스:** `/recruitment/member-cancel`
