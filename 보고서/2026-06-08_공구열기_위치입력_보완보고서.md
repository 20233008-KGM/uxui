## 1. vision.md 추가업무 반영

### 요청 내용
> **1. 공구열기 해도 위치 입력할 방법이 없어**

### 구현 내용

| 항목 | 내용 |
|------|------|
| **1단계 위치 입력** | `/open` 상단에 **수령 위치(팀장) 필수** 섹션 추가 |
| **현재 위치** | "현재 위치로 설정" 버튼 (GPS 시뮬레이션) |
| **동네 선택** | 신림동, 신림역, 봉천동, 남현동 프리셋 |
| **상세 주소** | 선택 입력 — 모집 확정 후 공개 안내 |
| **진행 제한** | 위치 미선택 시 **다음 단계 버튼 비활성화** |
| **2단계** | 등록 위치 요약 + "위치 수정하기" 링크 |
| **3단계** | 공유 위치·상세주소 최종 확인 |

**변경 파일:** `OpenGroupBuyPage.tsx`, `OpenGroupBuyStep2Page.tsx`, `OpenGroupBuyStep3Page.tsx`, `LocationUI.tsx`, `AppContext.tsx`

## 5. 확인 가이드

1. **위치 입력 (vision):** `/open` → 위치 선택 전 다음 단계 비활성 → 선택 후 활성
2. **현재 위치:** "현재 위치로 설정" 클릭
3. **2·3단계:** 위치 요약·상세주소 흐름 확인
4. **팀장 연락 두절:** `/leader/unavailable` 또는 상세 → 정책 → 링크
5. **알림 중복 방지:** `/mypage/notifications` → 토글 확인
