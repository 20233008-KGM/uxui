## 1. 프로젝트 이해 (vision.md)

| 항목 | 내용 |
|------|------|
| **목적** | 개인 구매가 아닌 **공동구매 단위**로 생필품을 더 저렴하게 구매 |
| **핵심 가치** | 동네 이웃 간 신뢰 기반 공구 + 가격 절감 |

## 2. 원시핵심유저플로우 구현 현황

### 2.1 진입 · 인증

| 플로우 단계 | 경로 | 구현 내용 |
|------------|------|----------|
| 랜딩페이지 | `/landing` | 서비스 소개, 공구페이·절약·알림 USP |
| 로그인 | `/login` | 이메일/비밀번호 로그인 → 메인 |
| 회원가입 | `/signup` | 닉네임·이메일·동네 등록 |
| 메인페이지 | `/` | 공구 목록, 배너, 하단 네비 (인증 필요) |

### 2.2 공구 열기 (팀장 플로우)

| 단계 | 경로 | 구현 내용 |
|------|------|----------|
| 1/3 품목 등록 | `/open` | 상품 사진·이름·카테고리·금액 |
| 2/3 수량·일정 | `/open/step2` | 최소/최대 인원, 마감일, 수령 일정, 수령 방법 |
| 3/3 결제 | `/open/step3` | 팀장 1인분 공구페이 결제, **수수료 안내** |
| 모집 결과 | `/recruitment/leader-success` | 모집 성공 → 채팅방 생성 안내 |

### 2.3 공구 참여 (참여자 플로우)

| 단계 | 경로 | 구현 내용 |
|------|------|----------|
| 근처 공구 탐색 (슬라이드) | `/explore/slide` | 카드 슬라이드 UI로 주변 공구 탐색 |
| 목록 탐색 | `/explore` | 검색·카테고리 필터 |
| 구매 이력 확인 | `/detail/:id` | **구매 이력 기반 재구매 추천** 배너 |
| 공구 세부사항 | `/detail/:id` | 환불·취소 정책, 도착 예정, 공구 정보 |
| 결제 | `/payment/:id` | 공구페이 안전결제 |
| 결제 완료 | `/payment/:id/complete` | 에스크로 보관 + 모집 현황 |
| 모집 성공 | `/recruitment/success` | 채팅방 입장 안내 |
| 모집 실패 | `/recruitment/failure` | 자동 환불 → `/refund/:txId` |

### 2.4 마이페이지

| 기능 | 경로 | 구현 내용 |
|------|------|----------|

## 3. edge_case.md 대응 기능

### 3.1 팀장 관련

| 예외 상황 | 대응 기능 |
|----------|----------|
| 먹튀 팀장 | 공구페이 에스크로 — 수령 확인 전 정산 불가 |
| 연락 두절 | 분쟁 신고 (`/dispute/:txId`) — 유형: 먹튀/연락 두절 |
| 주소 프라이버시 | 주소 관리 — 모집 확정 전 동 단위만 표시 |
| 수수료 부족 → 팀장 역할 기피 | 공구 열기 3단계 — **예상 수수료 수익** 명시 |

### 3.2 공구 진행 관련

| 예외 상황 | 대응 기능 |
|----------|----------|
| 최소 인원 미달 → 무산 | `/recruitment/failure` + `/refund/:txId` 자동 환불 |
| 결제 후 취소 | `/cancel/:txId` — 모집 완료 전 즉시 환불 |
| 불량·수량 부족 | 분쟁 신고 — 유형: 불량/수량 부족 |
| 허위 등록 | 분쟁 신고 — 유형: 허위 등록 |

### 3.3 수령 관련

| 예외 상황 | 대응 기능 |
|----------|----------|
| 수령 기간 내 미수령 | 수령 확인 화면 — 7일 기한·보관료·분쟁 절차 안내 |
| 팀장 보관 공간 부족 | 동일 안내 + 분쟁 신고 연결 |
| "내 것이 아니다" 분쟁 | 분쟁 신고 — 유형: 내 것이 아님 |

### 3.4 서비스 구조 관련

| 예외 상황 | 대응 기능 |
|----------|----------|
| 콜드스타트 | `ColdStartBanner` — 공구 열기·동네 범위 확장 유도 |
| 동네 범위 설정 | `/mypage/neighborhood` — 500m/1km/2km/동 단위 |
| 알림 주기 오류 | `/mypage/notifications` — 14일/7일/3일 전 설정, 중복 알림 방지 안내 |

## 5. 실행 방법

```bash
npm run dev
```

- URL: http://localhost:5173/
- 최초 진입: `/landing` → 회원가입/로그인 → 메인

### 추천 데모 플로우

1. **참여자:** 탐색 슬라이드 → 상세 → 공구페이 결제 → 모집 결과 → 채팅 → 수령 확인 → 후기
2. **팀장:** + 버튼 → 공구 열기 3단계 → 결제 → 모집 성공
3. **예외:** 상세 → 참여 취소 / 수령 확인 → 분쟁 신고 / 모집 실패 환불
