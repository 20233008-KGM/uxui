==========================================

[폴더 구성]
  deck.html          ← 발표 슬라이드 (이 파일을 더블클릭해서 열기)
  screens/           ← 슬라이드에 들어가는 앱 화면 이미지 11장
  발표_큐카드.md      ← 슬라이드별 발표 포인트 (스크립트 X, 키워드 O)
  README.txt         ← 지금 이 파일

------------------------------------------
[발표 슬라이드 여는 법]
  1) deck.html 더블클릭 → 기본 브라우저(크롬 권장)에서 열림
  2) 키보드 조작
       → / ← / Space : 다음 / 이전 슬라이드
       F            : 전체화면 (발표 시 필수)
       Esc          : 전체 슬라이드 한눈에 보기(오버뷰)
       S            : 발표자 노트 모드
  * 인터넷 연결 필요 (폰트·슬라이드 엔진을 CDN에서 불러옴)

------------------------------------------
[게시판 업로드용 PDF 만드는 법]  ※ 6/15(월) 12:00까지 업로드 필수
  1) deck.html을 크롬으로 연다
  2) 주소창 끝에  ?print-pdf  를 붙이고 Enter
       예) file:///.../deck.html?print-pdf
  3) Ctrl+P (인쇄)
  4) 대상=PDF로 저장, 용지=가로(가로방향),
     여백=없음, 배경 그래픽 체크 ON
  5) 저장 → 게시판 [21. 기말프로젝트 발표자료]에 업로드
